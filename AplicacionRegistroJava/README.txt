Para un correcto funcionamiento de la aplicación copie la carpeta DB en C:\Users\Public\Documents.

Con esta aplicación podremos: 
	- Registrarnes en una base de datos. 
	- Iniciar sesión para consultar nuestros datos.
	- Imprimir nuestros datos en un fichero TXT.
	- Eliminar nuestos datos de la base de datos.
